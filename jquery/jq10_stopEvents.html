이벤트처리막기
    1) e.preventDefault() : 고유동작 중단
    2) e.stopPropagation() : 이벤트상위전파 중지
    3) e.stopImmediatePropagation() : 이벤트가 상위뿐 아니라 같은 레벨로도 전파 X
    4) return false;
        jQuery사용시 : e.stopPropagation() + e.preventDefault() 동작하지않고 상위로 전파도 안됨
        jQuery미사용시 : e.preventDefault() 고유동작만중단
